Bootstrap3 Datepicker

a fork of bootstrap-datepicker for use with Bootstrap3

Usage

Attached to a field with the format specified via options

<input type="text" value="02-16-2012" class="datepicker-attach">

$('.datepicker-attach').datepicker({
    format: 'mm-dd-yyyy'
});

Attached to a field with the format specified via options

<div value="02-16-2012" class="datepicker-inline"><div>

$('.datepicker-inline').datepicker({
    format: 'mm-dd-yyyy'
});
